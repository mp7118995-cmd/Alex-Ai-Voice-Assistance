# Offline AI Voice Assistant

An Offline AI Voice Assistant powered by Ollama that performs daily tasks using natural voice commands.

## Features

- Offline AI using Ollama
- Voice Recognition
- Text-to-Speech
- WhatsApp Automation
- Open Applications
- Close Applications
- AI Chat
- Local Processing

## Installation

```bash
git clone https://github.com/mp7118995-cmd/Alex-Ai-Voice-Assistance.git



pip install -r requirements.txt
```

Install Ollama and download a model.

Example:

```bash
ollama pull llama3.2
```

Run the assistant

```bash
python main.py
```
